# Ansible Nginx Multi-Server Deployment

This project demonstrates deploying a static HTML website to **three Linux servers**
using **Ansible**.

## What I Built
- Ansible control node
- 3 managed servers
- Nginx installed and managed via Ansible
- Same website deployed to all servers

## Tech Stack
- Ansible
- Nginx
- Linux
- SSH key authentication
